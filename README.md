# 🌌 Nebula Haven – Online Cosmetic Shop

Nebula Haven is a full-stack role-based e-commerce web application designed for buying and managing cosmetic products. It provides a seamless shopping experience for customers while offering powerful management tools for administrators.

---

## 🚀 Features

### 👤 Customer Features
- Browse cosmetic products with a clean UI
- Add products to cart and manage purchases
- Secure login and authentication (JWT-based)
- Place orders and track order status
- Receive notifications for order updates

### 🛠️ Admin Features
- Admin dashboard for system management
- Manage products (Add / Update / Delete)
- Manage orders and track customer activity
- Manage users and roles

## 🔐 Authentication Flow

1. User logs in with credentials
2. Server verifies password (hashed comparison)
3. JWT token is generated
4. Token is sent to client
5. Client sends token in Authorization header for protected routes
